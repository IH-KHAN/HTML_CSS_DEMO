# 🏦 Financial & Banking Front-End Portal

A comprehensive, multi-page static web application built purely with **HTML5 and CSS3**. Themed around corporate banking and finance, this project serves as both a demonstration of modern front-end design principles and a structured, chapter-by-chapter educational template (spanning 22 distinct chapters/modules).

## 🎨 Architecture & UI Design

* **Static Site Architecture:** A lightweight, blazing-fast front-end requiring no backend server or database to run. 
* **Centralized Styling:** Utilizes a single, globally linked stylesheet (`banking_style.css`) to ensure consistent branding, typography, and layout across all 20+ HTML pages.
* **Multimedia Integration:** Native HTML5 implementation of audio (`speech.mp3`), scalable vector and raster graphics, and downloadable document links (e.g., `Audit_Report_2025.pdf`).
* **Modular Progression:** The codebase is structured sequentially (`chapter_01.html` through `chapter_22.html`), making it an excellent reference for progressive web design concepts or a multi-step user onboarding flow.

## 📂 Key Features & Directory Structure

### 🌐 Core Pages
* **`Homepage.html`:** The main landing page acting as the navigational hub for the corporate portal.
* **`chapter_01.html` - `chapter_22.html`:** Individual modules/pages demonstrating various HTML/CSS layouts, UI components, and financial data presentations.

### 🖼️ Asset Management
The project strictly follows best practices for separating assets into dedicated directories:
* **`/icon/`:** Contains thematic UI elements (e.g., `business-report.png`, `investment.png`, `locked.png`) and CSS Sprite sheets for optimized loading.
* **`/images/`:** Houses larger graphical assets, charts (e.g., `vix-chart-1990-2013.png`, `Assets-and-Liabilities_.png`), and modern `.webp` formats.
* **`/media/`:** Contains embedded multimedia files like audio streams (`speech.mp3`) and video thumbnails.
* **`/files/`:** Stores secure or downloadable corporate documents, such as the 2025 Audit Report.

## 🛠️ Technical Stack

* **Markup:** HTML5 (Semantic elements, native media embedding)
* **Styling:** CSS3 (Custom layouts, sprite mapping, typography)
* **Assets:** WebP, PNG, JPG, MP3, PDF

## 🚀 Getting Started

Because this is a purely static front-end project, there are no dependencies, package managers, or local servers required to view the application.

### Prerequisites
* Any modern web browser (Google Chrome, Mozilla Firefox, Microsoft Edge, Safari, etc.)
* A code editor (like VS Code) if you wish to modify the source code.
