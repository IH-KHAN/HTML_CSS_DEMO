18.4 CSS Overflow (Scrolling Data)

Manages content that is too big for its container, essential for transaction logs.

Visible: Content spills out. Transaction #1, Transaction #2, Transaction #3, Transaction #4...
Hidden: Content clipped. Transaction #1, Transaction #2, Transaction #3, Transaction #4...
Scroll: Always scrollbars. Transaction #1, Transaction #2, Transaction #3, Transaction #4...
Auto: Scrollbars if needed. Transaction #1, Transaction #2, Transaction #3, Transaction #4...
