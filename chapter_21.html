21.1 CSS Variables (Global Theming)

CSS Variables (Custom Properties) allow us to define brand colors and spacing in one place (`:root`) and reuse them everywhere. This is critical for maintaining brand consistency across a large banking platform.

Standard Account Card

This card uses `var(--primary-brand)` for its border and text color, and `var(--card-padding)` for spacing.


If we change `--primary-brand` in the CSS file, every single card on the website updates instantly.

21.2 CSS Calculation (`calc()`)

The `calc()` function performs calculations to determine CSS property values. It is essential for mixing units (e.g., pixels and percentages) to create fluid, responsive dashboard layouts.

Sidebar
Width: 200px (Fixed)
Main Content Area
Width: calc(100% - 200px)

This area automatically fills the remaining space, ensuring the layout never breaks regardless of screen size.

21.3 CSS Counters (Legal Document Numbering)

CSS Counters allow us to adjust the appearance of content based on its location in the document. This is perfect for automatically numbering sections and clauses in long Terms of Service or Audit Reports.

Terms of Service

User Responsibilities.

Account Security Mandates.

Liability Limitations.

Data Collection.

Arbitration Process.
