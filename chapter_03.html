3.1 Formatting Elements (Document Emphasis)

Formatting elements are used to give text specific meaning or visual emphasis, critical for clarity in financial agreements and reports.

3.1.1 Bold (`<b>`) - Visual Emphasis

Used to draw attention without implying strong importance (semantic importance uses `<strong>`).

Principal Amount: The original sum of money borrowed in a loan or put into an investment.

3.1.2 Italic (`<i>`) - Alternate Voice

Used for technical terms, foreign phrases, or thoughts.

The term force majeure refers to unforeseeable circumstances that prevent the fulfillment of a contract.

3.1.3 Underline (`<u>`) or Insert (`<ins>`)

`<u>` renders a simple underline. `<ins>` indicates text that has been added to a document (useful for policy updates).

Standard Checking Account terms apply.
Updated Policy: Minimum balance requirement waived for students.

3.1.4 Strikethrough (`<s>`) or Delete (`<del>`)

Used to show data that is no longer accurate or has been removed.

Annual Fee: $120.00 $0.00 (Waived)
Old Interest Rate: 2.5% (Expired)

3.1.5 Highlight (`<mark>`) - Key Information

Highlights text for reference or importance, like highlighting a search term or a critical figure.

Please note that the Interest Rate is variable and subject to change based on the prime rate.

3.1.6 Subscript (`<sub>`) and Superscript (`<sup>`)

Essential for mathematical formulas in financial calculations and footnotes.

Compound Interest Formula: A = P(1 + r/n)nt
Risk Coefficient (Beta): βi

3.2 HTML Style Attribute (Inline Styling)

The `style` attribute allows you to apply CSS directly to an element. While external stylesheets are preferred, inline styles are useful for quick, local overrides.

SecureBank Portal

This paragraph uses inline styles to set font size, text color, background color, and padding directly within the HTML tag.
