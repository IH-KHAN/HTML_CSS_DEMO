17.1 CSS Background Properties

17.1.1 background-color

Sets the base color for an element. Used for card backgrounds and sections.

Account Overview Section (Light Blue Background)

17.1.2 background-image

Sets an image or pattern. Often used for subtle security watermarks.


17.1.3 background-repeat

Controls if/how the image repeats.


17.1.4 background-position

Positions the image (e.g., center, top-right). Essential for aligning icons.


17.1.5 background-size

Controls the size of the image relative to its container.

Contain (Fit)
Cover (Fill)

17.1.6 background (Shorthand)

Combines color, image, repeat, position, and size in one line. 

background: #ffffff url('icon.png') no-repeat center / contain;

17.1.7 background-clip

Defines how far the background extends (border-box, padding-box, content-box).

Content Box Clip: The blue background stops at the padding edge.

17.1.8 background-origin

Specifies where the background image is positioned relative to the box model.

Border Box Origin: Image starts at the top-left of the border.

17.1.9 background-attachment

Determines if the background scrolls with the content (`local`) or stays fixed (`fixed`).

Scroll this box.
(Security Watermark Scrolls)

17.2 CSS Gradient Backgrounds

17.2.1 Linear Gradients

Creates a smooth transition along a line. Used for modern headers and buttons.

SecureBank Header (Blue to Green)

17.2.2 Repeating Linear Gradients

Creates striped patterns, often used for "Draft" or "Confidential" overlays.


17.2.3 Radial Gradients

Radiates from a center point. Can create a "spotlight" effect.


17.2.4 Repeating Radial Gradients

Creates circular patterns (rarely used in clean financial UIs).

17.3 CSS Sprites (Performance Optimization)

CSS Sprites combine multiple small icons into a single image file. We then use `background-position` to show only the specific icon needed. This reduces server requests and speeds up dashboard loading.
